<h3 id="헤더" style="text-align:center; color:#4a6a8a; font-weight:700; margin-bottom:12px;">STEP 1. 전표 헤더 입력</h3>

<div style="background:#f7f9fb; border:1px solid #eceef1; border-radius:10px; padding:16px 20px; margin-bottom:10px; font-size:0.93em; color:#333;">
  <div style="margin-bottom:8px;">T-CODE : <strong style="color:#4a6a8a;">F-63</strong> &nbsp;(전표 임시전기)</div>
  <ul style="margin:0; padding-left:1.4em; line-height:2;">
    <li><strong>증빙일 = 전기일 = 기간(月)</strong> 동일하게 기재</li>
    <li>참조 : <strong>부서명 / 작성자이름</strong> &nbsp;<span style="color:#888; font-size:0.9em;">(예: 회계팀/홍주형)</span></li>
    <li>전기키 <strong>31</strong>(대변_미지급금) / 계정 : <strong>7 + 사번</strong> 입력 후 Enter</li>
    <li style="color:#888; font-size:0.9em;">※ 조회 안 되는 경우 보상운영팀 문의</li>
  </ul>
</div>

<h3 id="개요" style="text-align:center; color:#3d6e47; font-weight:700; margin-bottom:12px;">STEP 4. 개요 확인 후 저장</h3>

    <li>차변(D) 합계 = 대변(C) 합계 확인</li>
